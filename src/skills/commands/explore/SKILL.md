# Explore Command

When this skill is invoked (e.g., via `/explore <topic>`), use the `tech-explorer` skill to:

1. Perform deep research using web search and official docs.
2. Create a test scaffold to empirically verify functionality.
3. Identify and document pitfalls for the 'Wall of Shame'.
4. Generate a 'Gold Standard' SKILL.md for future use.
