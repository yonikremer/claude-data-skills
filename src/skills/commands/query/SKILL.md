# Query Command

When this skill is invoked (e.g., via `/query <task>`), use the `database-pro` skill to:

1. Write or optimize the SQL query for the given task.
2. Follow master-level SQL patterns: use CTEs for readability, window functions for analytics, and ensure the query is
   optimized.
