# Analyze Command
When this skill is invoked (e.g., via `/analyze <file>`), use the `exploratory-data-analysis` skill to:
1. Detect grain, structure, and nullity patterns.
2. Identify categorical hierarchies and redundant columns.
3. Generate an EDA report and update the data dictionary.
