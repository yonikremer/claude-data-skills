# Claude Code Data Analytics Skills 

The purpose of this codebase is to collect and manage the Claude Code skills I will use daily. All skills are now organized in the `skills/` directory.

## Core Principles

- **Data Safety First**: NEVER delete, overwrite, or drop data without explicit permission.
- **Resource Aware**: Every intensive task starts with a hardware resource check.
- **OS Model Optimized**: Skills are dense, idiomatic, and contain strict guardrails for local/open-source LLMs.
- **Modern Visualization**: Plotly is preferred over Matplotlib for all high-quality outputs.
