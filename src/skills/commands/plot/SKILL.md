# Plot Command

When this skill is invoked (e.g., via `/plot <type> <file>`), use the `visualization-pro` skill to:

1. Create a plot of the specified type for the data in the provided file.
2. Prefer Plotly for interactivity.
3. Ensure the code is production-ready with proper labels and accessibility themes.
